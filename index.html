How it works

  • Create or join a room with a code.
  • Messages update instantly for everyone in the room.
  • Typing indicator shows when others are composing.
Firebase Realtime Database Rules (required)
{
  "rules": {
    "rooms": {
      "$room": {
        ".read": "auth != null",
        ".write": "auth != null"
      }
    }
  }
}
